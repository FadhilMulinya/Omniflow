# ⚙️ Omniflow API KIT

Omniflow helps developers connect and work with multiple APIs using a simple drag-and-drop interface. You can also build AI agents, but the core focus is making API workflows faster, easier, and more intuitive.

We’re also building **custom components for non-technical users**, so they can work with APIs and AI tools without writing code.

---

## 🚀 What You Can Do

- Drag and drop to build custom workflows  
- Connect and interact with multiple APIs  
- Create lightweight AI agents  
- Build reusable custom components  
- Skip boilerplate, focus on logic  

---

## 🧩 What’s Coming

We’re working on a suite of products alongside the Omniflow Agent:

- **Omniflow Studio**: A visual playground to build and deploy workflows  
- **Omniflow Components**: Pre-built tools for non-devs (think Airtable meets AI)  
- **Omniflow Runtime**: A lightweight, pluggable execution layer  
- **Omniflow Templates**: Ready-to-use API + AI workflow packs
